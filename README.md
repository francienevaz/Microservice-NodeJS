# Microservice-NodeJS

Projeto desenvolvido durante algumas lives, dentro da plataforma da DIO, é um microserviço de autenticação que poderá ser muito útil no seu dia a dia.

Endpoints base do projeto:

Usuários

* GET /users

* GET /users/:uuid

* POST /users

* PUT /users/:uuid

* DELETE /users/:uuid


Autenticação

* POST /token

* POST /token/validate
